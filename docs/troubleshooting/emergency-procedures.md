# Emergency Procedures

Critical failure handling for when things go wrong during service. Stay calm and follow these steps.

## Critical Audio Failure

### Total Audio Loss

**When:** No sound at all, congregation can't hear anything

**Immediate Actions:**
1. Check if Power Conditioner is on (stage rack)
2. Check if X18 mixer has lights (should see green level meters)
3. Check if main output is muted in X-Air Edit

**If nothing works:**
1. Announce to congregation: "Technical difficulty, please wait one moment"
2. Call tech support: [+1 917-664-1081](tel:+19176641081)
3. Consider alternative: Use portable PA system if available
4. If no fix possible: Ask speaker to project voice louder, move closer to congregation

### Emergency Mute All

**When:** Severe feedback that won't stop, or need to immediately silence all audio

**How to execute:**
- **Option 1 (Fastest):** Companion Page 1 → "MUTE ALL EMERGENCY" button (bottom of row 4)
- **Option 2:** X-Air Edit → Main output → Click MUTE button
- **Option 3:** Power Conditioner → Turn OFF (last resort)

**After muting:**
1. Identify the problem source (which mic, which monitor)
2. Lower volume or move equipment
3. Slowly unmute and test
4. Resume service

## Critical Video Failure

### Projector Completely Failed

**When:** Projector won't turn on, no image, or physical damage

**Immediate Actions:**
1. Try remote control (new batteries)
2. Check projector power cable
3. Try manual power button on projector if accessible

**If nothing works:**
1. Inform leadership: "Projector is down, continuing without slides"
2. ProPresenter operator: Read slide content aloud if needed
3. Worship leader: Know songs by heart or use printed lyrics
4. Call tech support after service for repair

## Safety Emergencies

### Smoke or Burning Smell

**IMMEDIATE ACTIONS:**
1. **Turn off Power Conditioner immediately**
2. **Unplug suspicious equipment**
3. **Evacuate if necessary**
4. **Call 911 if fire visible**

**Do NOT:**
- Continue using smoking equipment
- Ignore burning smells
- Touch hot equipment

## Contact During Emergencies

### Tech Support Emergency Line
- **Phone:** [+1 917-664-1081](tel:+19176641081)
- **When to call:** Critical failures during service
- **Have ready:** Clear description of what failed and what you tried

### Emergency Services
- **911:** Fire, medical emergency, immediate danger

## Remember

::: tip Stay Calm
The most important thing in an emergency is to stay calm. Service can continue in most situations, even with technical difficulties. The congregation understands, and leadership will adapt.
:::

::: warning Safety First
**Never risk safety for technical perfection.** If equipment shows signs of danger (smoke, sparks, burning smell), turn it off immediately and get help. No service is worth risking injury or fire.
:::
