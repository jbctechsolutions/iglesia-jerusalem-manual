## Equipment Specifications

### Audio Equipment
- **Mixer:** Behringer X18 Digital Mixer
- **Wireless Mics:** Shure wireless system (4 handhelds, 4 lapels)
- **Main Speakers:** JBL EON615 (2x 15" powered)
- **Stage Monitors:** Yamaha DBR10 (2x 10" powered)

### Video Equipment
- **Camera:** PTZOptics 12X-NDI camera with PoE
- **Projector:** BenQ ceiling-mounted projector
- **Recording:** OBS Studio on Mac Mini
- **Streaming:** Zoom with NDI integration

### Control Systems
- **Computer:** Mac Mini M2 (Production workstation)
- **Network:** Ubiquiti UniFi network infrastructure
- **Power:** Furman power conditioner with smart plug control
- **Automation:** ij-control system for automated startup/shutdown
